## Quick Customization Guide

### Change Site Name
```
Search in all files: "Ridge & Rowan"
Replace with: "Your Clinic Name"
```

### Change Colors
```
In <style> section, find :root{ }
Change --primary: #2b9cda to your color
Change --accent: #00d9a3 to your accent
```

### Update Contact Info
```
Find contact section in index-medical.html
Update phone number, email, address, hours
```

### Add New Service Page
```
1. Copy services-implants.html
2. Rename to services-yourservice.html
3. Update title, content, images
4. Add link from main page
```
